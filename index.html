API Mashup

CONSUMO DE 6 CATEGORÍAS DE APIs

Un Mashup es una aplicación web que combina datos y funciones de múltiples fuentes externas en una sola interfaz. Este proyecto integra 6 tipos distintos de APIs reales funcionando al mismo tiempo dentro de un solo dashboard interactivo.

COMPONENTES DEL PROYECTO
Las 6 APIs integradas
📍
API · 01
Geolocalización
Obtiene las coordenadas GPS reales del dispositivo usando la API nativa del navegador. Muestra latitud, longitud y precisión en metros sobre un mapa SVG interactivo.
navigator.geolocation
🌐
API · 02
Redes Sociales
Simula la extracción de perfiles públicos conectando con REST Countries API. Muestra información de países como capital, idioma, moneda y población en formato de feed social.
restcountries.com/v3.1
🛒
API · 03
E-Commerce
Consulta un catálogo real de productos con imagen, precio en USD y calificación de usuarios. Filtra por categorías como electrónica, joyería y ropa usando FakeStore API.
fakestoreapi.com
🗄️
API · 04
Base de Datos
Envía un formulario de registro a la nube mediante una petición HTTP POST real. El servidor responde con un ID único asignado al nuevo usuario, visible en la terminal de logs.
jsonplaceholder.typicode.com
📱
API · 05
Protocolo SMS
Simula el flujo completo de envío de notificaciones usando el protocolo de Twilio. Soporta SMS, Push notifications, WhatsApp y Email con historial de mensajes en tiempo real.
Twilio API (simulación)
▶️
API · 06
Streaming
Incrusta y controla un reproductor de YouTube usando la IFrame Player API. Permite reproducir, pausar, detener y silenciar videos en tiempo real mediante comandos JavaScript.
YouTube IFrame API
FUNCIONAMIENTO
¿Cómo funciona un Mashup?
01
SOLICITUD
El navegador hace una petición HTTP a una API externa usando fetch()
02
RESPUESTA
La API devuelve datos en formato JSON con la información solicitada
03
PROCESAMIENTO
JavaScript procesa y transforma los datos para mostrarlos en la interfaz
04
VISUALIZACIÓN
El HTML se actualiza dinámicamente sin recargar la página completa
